## 技术预研清单

研发正式启动前需完成以下预研，结论影响架构决策：

1. **NBD 虚拟 FAT32 性能预研**（R01）：在真实 RK3568 测试 NBD 虚拟 FAT32 大文件读写延迟和 I/O error 阻断效果
2. **ClamAV 性能预研**（R02）：在真实 RK3568 测试大容量 U 盘全盘扫描耗时
