# 04 前端页面与交互设计映射

页面来源说明：PRD = 来自 PRD 定义，原型 = 原型图有体现，两者都有 = PRD 和原型均有。

---

## G01 全局页面交互规则

| 项目 | 内容 |
|---|---|
| 管理端数据原则 | 管理端不持久化业务数据、账号数据、日志数据、最近连接 IP、当前页面进程内状态或查询结果；仅在进程内存中保存当前连接、当前会话、当前页面状态和当前请求结果 |
| 会话失效 | 管理端 5 分钟无操作自动登出，并使当前内存会话失效；再次使用需重新输入装置 IP、用户名和密码 |
| 管理端关闭/重启 | 管理端关闭、崩溃或重启后，所有进程内存状态清空；再次使用需重新输入装置 IP、用户名和密码 |
| 主动登出 | 用户点击登出后，管理端调用装置侧登出指令失效当前会话 token；调用完成或连接已不可用时，管理端均清空当前进程内会话状态并返回登录页 |
| 装置连接断开 | 管理端与装置断开连接后，不强制退出当前页面；页面保留当前页面进程内状态，用户点击查询、导出、导入、保存、删除、升级、授权等下发动作时提示操作失败 |
| 重连处理 | 重新连接后，管理端向装置侧校验当前内存会话；会话有效时停留当前页面并刷新装置侧真实数据，会话失效时返回登录页重新登录 |
| 权限控制 | 菜单和按钮按装置侧返回的当前用户角色展示；装置侧接口仍必须校验会话状态、授权状态、角色权限和操作合法性 |
| 请求超时 | 普通指令 15 秒超时，查询类自动重试 1 次；文件传输类（策略导入/导出、系统升级、病毒库升级、日志导出、授权文件上传）300 秒超时，不自动重试；超时后提示用户 |
| 耗时操作 | 文件传输类指令发送后操作按钮 disable + loading，禁止重复提交；超时且连接仍在时提示"操作超时，装置可能仍在处理，请稍后确认结果再决定是否重试"；超时且连接断开时提示"网络连接中断，请重新连接后查看装置状态" |

## P01 登录页

| 项目 | 内容 |
|---|---|
| 页面路径 | `/login` |
| 页面来源 | 两者都有 |
| 页面目标 | 用户输入装置 IP、用户名、密码，连接装置并登录；已登录用户可主动登出并返回登录页 |
| 主要区域 | Logo 区域、表单区域、登录按钮 |
| 用户可执行操作 | 填写装置 IP / 用户名 / 密码，点击登录；登录后从用户菜单点击登出 |
| 表单字段 | 装置 IP（必填，IPv4格式校验）/ 用户名（必填）/ 密码（必填） |
| 前端状态 | 默认、提交中（按钮 loading）、登录失败（错误提示）、账号锁定（锁定提示） |
| 接口依赖 | C03 连接装置，S10/S08 装置侧登录鉴权、授权状态与角色返回；登出时调用 S10/S08 登出接口 |
| 权限要求 | 无需登录 |
| 加载态 | 点击登录按钮后 loading，防止重复提交 |
| 空状态 | 不适用 |
| 错误态 | 装置IP格式错误 / 装置连接失败 / 用户名或密码错误 / 账号锁定提示 |
| 自动化测试点 | 装置IP为空禁止提交，IP格式错误提示，登录页 IP 输入框为空且不展示默认 IP，5次失败后锁定提示，锁定期间输入正确密码仍提示锁定，登录成功失败计数清零，主动登出后清空进程内会话状态并返回登录页 |
| 原型与PRD差异 | 原型存在 IP 示例值；PRD 要求登录页 IP 输入框为空，不预填、不记住、不展示默认 IP |

| 原型与PRD差异 | 一致 |

## P03 文件访问控制页

| 项目 | 内容 |
|---|---|
| 页面路径 | `/file-access` |
| 页面来源 | 两者都有 |
| 页面目标 | 配置三种文件访问控制策略 |
| 主要区域 | 可执行程序访问控制卡片 / 介质自动读取功能控制卡片 / 文件类型访问控制卡片（含黑名单表格） |
| 用户可执行操作 | 勾选/取消勾选三个开关（实时写入装置）/ 添加文件类型黑名单 / 删除黑名单条目（二次确认） |
| 表单字段 | 黑名单添加：文件后缀（必填，.开头，格式校验）/ 说明文字 |
| 前端状态 | 已连接/未连接（连接断开时保留当前页面进程内状态，下发动作失败提示）/ 三个开关各自的勾选状态 |
| 接口依赖 | C03 调用装置接口，C04 装配页面数据，S04 写入开关状态和黑名单变更 |
| 权限要求 | 操作员 |
| 加载态 | 进入页面时读取装置当前配置 loading |
| 空状态 | 黑名单无条目时展示"暂无数据" |
| 错误态 | 管理端与装置断开连接时点击下发动作提示失败 / 后缀格式错误 / 重复后缀提示 / 写入失败提示 |
| 自动化测试点 | 连接断开时不跳登录页且当前页面进程内状态保留，点击开关/新增/删除提示下发失败，添加空后缀拒绝，重复后缀拒绝，删除二次确认，写入成功提示重新拔插生效 |
| 原型与PRD差异 | 可执行访问控制内置类型按 PRD 为 4 个（dll / exe / PE / ELF，基于文件实际格式识别）；文件类型访问控制内置黑名单为 38 个，删除与可执行控制重复项 |

## P04 U盘设备控制页

| 项目 | 内容 |
|---|---|
| 页面路径 | `/usb-devices` |
| 页面来源 | 两者都有 |
| 页面目标 | 管理普通U盘白名单，设置读写权限 |
| 主要区域 | 白名单列表（操作列含修改/删除按钮）/ 装置端添加按钮 / 管理端添加按钮 |
| 用户可执行操作 | 装置端添加（弹窗）/ 管理端添加（弹窗）/ 修改（弹窗：权限 + 描述）/ 删除（二次确认） |
| 列表展示字段 | 序列号 / 描述 / 权限（只读、读写）/ 添加方式（装置端添加、管理端添加）/ 添加时间 / 操作 |
| 表单字段 | 添加弹窗：设备选择（序列号/设备名称）/ 描述（可选）/ 权限（只读/读写，必选）；修改弹窗：权限（只读/读写）/ 描述（可选） |
| 添加数据源 | 装置端添加的数据源为当前连接到 USB 管控装置且 `admission_status=addable` 的普通大容量存储设备；管理端添加的数据源为管理主机本机 USB 大容量存储设备枚举结果 |
| 内部不展示字段 | VID / PID / 设备名称 / 容量 / 设备类型，由装置侧保存并用于设备识别和审计 |
| 前端状态 | 已连接/连接断开（连接断开时保留当前页面进程内状态，下发动作失败提示） |
| 接口依赖 | C03 调用装置接口，C04 装配白名单列表，S05/S10 读取白名单并执行增删改 |
| 权限要求 | 操作员 |
| 加载态 | 进入页面读取白名单 loading / 添加弹窗打开时读取接入设备 loading |
| 空状态 | 白名单无条目时展示"暂无数据" |
| 错误态 | 未检测到U盘 / 序列号为空禁止添加 / 已存在提示 / 写入失败提示 |
| 自动化测试点 | 序列号为空不可添加，重复序列号提示，删除后该设备视为未认证，权限和描述修改记录装置侧操作日志，连接断开时点击新增/修改/删除提示下发失败 |
| 原型与PRD差异 | 原型只有删除按钮；按 PRD 补充"修改"按钮，点击弹窗同时支持修改权限和描述 |

| 原型与PRD差异 | 原型有"安全U盘自动升级"按钮，PRD本版本不支持，不展示该按钮 |
