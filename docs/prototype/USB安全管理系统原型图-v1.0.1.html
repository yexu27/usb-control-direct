介质自动读取指对U盘中autorun配置文件进行控制。启用=含.sh文件的U盘映射后，.sh文件内容显示为空（不可读）；禁用=.sh文件可正常读写。仅对U盘根目录下的.sh后缀文件生效。默认：未勾选（禁用）
